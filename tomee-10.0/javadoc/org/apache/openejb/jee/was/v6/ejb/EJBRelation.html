Class EJBRelation

  • java.lang.Object
    • org.apache.openejb.jee.was.v6.ejb.EJBRelation

  • public class EJBRelation
extends java.lang.Object
    Since:
    J2EE1.3 The ejb-relation element describes a relationship between two entity beans with container managed persistence. An ejb-relation element contains a description; an optional ejb-relation-name element; and exactly two relationship role declarations, defined by the ejb-relationship-roles. The name of the relationship, if specified, is unique within the ejb-jar file.
