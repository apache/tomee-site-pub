Class JndiEncInfo

  • All Implemented Interfaces:
    java.io.Serializable

    public class JndiEncInfo
extends InfoObject
    JndiEncInfo holds a context-worth of jndi information. The info objects are normalized so their names include the comp, module, app, or global part of the jndi path but not the java: namespace. Each instance should contain only info objects for one of the subcontexts.
