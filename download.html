Downloads

Release Integrity

You must verify the integrity of the downloaded files. We provide OpenPGP signatures (*.asc files) for every release file. This signature should be matched against KEYS file which contains the OpenPGP keys of TomEE’s Release Managers. We also provide SHA-512 checksums for every release file. After you download the file, you should calculate a checksum for your download, and make sure it is the same as ours.
