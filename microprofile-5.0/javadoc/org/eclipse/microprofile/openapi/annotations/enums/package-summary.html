Package org.eclipse.microprofile.openapi.annotations.enums Description

A set of parameters used by other annotations to represent their enumerable properties. This package contains enums for the following annotations:
  • Parameter: in and style properties
  • Schema: type property
  • SecurityScheme: in and type properties
  • various annotations that have explode property
