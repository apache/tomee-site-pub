      • getOtherTestResource

        protected java.lang.Object getOtherTestResource(TestResource resourceConfig)
        Override to perform custom resource types injection. This method will be called when whatever value was specified in the TestResource annotation wasn't understood by the performInjections(java.lang.Object) method. This method will be called, supplying the annotation, and you can then interpret and create the value to be injected. By default this method just returns null.

        You can use this to inject values into annotated fields which contain custom values in their names.
