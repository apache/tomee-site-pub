  • @Qualifier
 @Documented

public @interface Named
    String-based qualifier.

    Example usage: 

       public class Car {
     @Inject @Named("driver") Seat driverSeat;
     @Inject @Named("passenger") Seat passengerSeat;
     ...
   }
