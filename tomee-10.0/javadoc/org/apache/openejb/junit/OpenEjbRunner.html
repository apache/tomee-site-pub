      • newTestContext

        public TestContext newTestContext(java.lang.reflect.Method method,
                                  java.lang.String roleName)
        This accepts a null method. Might make it easier to initialize new contexts from the delegate runner (which calls back here). The cohesion isn't that great but is the best I could come up with for delegating to both JUnit 3 and 4. The alternative was a context factory - though that introduced a new class and made extending it more complex as 3 classes are needed - a runner to identify the factory, a factory and a context. Either that or you need to specify the factory class in every test.
