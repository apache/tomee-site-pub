jakarta.xml.ws.spi

Class Invoker

  • java.lang.Object
    • jakarta.xml.ws.spi.Invoker

  • public abstract class Invoker
extends java.lang.Object
    Invoker hides the detail of calling into application endpoint implementation. Container hands over an implementation of Invoker to Jakarta XML Web Services runtime, and Jakarta XML Web Services runtime calls invoke(java.lang.reflect.Method, java.lang.Object...) for a web service invocation. Finally, Invoker does the actual invocation of web service on endpoint instance. Container also injects the provided WebServiceContext and takes care of invoking jakarta.annotation.PostConstruct methods, if present, on the endpoint implementation.
    Since:
    1.7, JAX-WS 2.2
    See Also:
    Provider.createEndpoint(String, Class, Invoker, WebServiceFeature...)
