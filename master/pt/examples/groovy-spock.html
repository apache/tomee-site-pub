Projeto Groovy com Spock Framework

Este é um guia de como implementar Spock Framework para testes.

HelloSpecification

Esta classe estende de spock.lang.Specification para registrar a especificação de testes.

Injete o bean Hello com @Inject e use Arquilian para desdobrar o teste.
