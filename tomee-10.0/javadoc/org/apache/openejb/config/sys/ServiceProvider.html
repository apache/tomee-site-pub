      • getProperties

        public java.util.Properties getProperties()
        Gets the value of the properties property. This accessor method returns a reference to the live Properties Object, not a snapshot. Therefore any modification you make to the returned Properties will be present inside the JAXB object. This is why there is not a set method for the properties property. For example, to add a new value, do as follows: 
            getProperties().setProperty(key, value);
