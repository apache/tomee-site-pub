Interface Assembler

  • All Known Implementing Classes:
    Assembler

    public interface Assembler
    Instantiates and assembles a configured, runnable, instance of the container system and all of its sub-components. Vendors needing extreme control over the construction of the container system can get it by implementing this class. Doing this comes with large amounts of responsibility and complexity and should not be done without a deep understanding of OpenEJB. This class responsbilities are:
    • Instantiate and initialize all Container implementations
    • Instantiate and initialize TransactionService implementation
    • Instantiate and initialize SecurityService implementation
    • Instantiate and initialize all ResourceManagers
    • Load all deployed beans
    • Populate each deployment's JNDI ENC
    • Populate the IntraVM Server's global, client, JNDI namespace

    For the concrete implementation please also inspect Assembler.
