Class Slf4jLogger

  • java.lang.Object

        • org.apache.openejb.log.logger.Slf4jLogger

  • public class Slf4jLogger
extends AbstractDelegatingLogger

    java.util.logging.Logger implementation delegating to SLF4J.

    Methods Logger.setParent(Logger), Logger.getParent(), Logger.setUseParentHandlers(boolean) and Logger.getUseParentHandlers() are not overrriden.

    Level mapping inspired by org.slf4j.bridge.SLF4JBridgeHandler: 

     FINEST  -> TRACE
 FINER   -> DEBUG
 FINE    -> DEBUG
 CONFIG  -> DEBUG
 INFO    -> INFO
 WARN ING -> WARN
 SEVER   -> ERROR
